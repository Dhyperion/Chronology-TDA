\doxysubsection{Descripción detallada}
T.\+D.\+A. de un conjunto de \mbox{\hyperlink{class_fecha_historica}{Fecha\+Historica}} ordenadas por fechas. 

una instancia {\itshape c} del tipo de datos abstracto {\ttfamily \mbox{\hyperlink{class_cronologia}{Cronologia}}} es un objeto que agrupa acontecimientos ocasionados en distintas fechas, compuesto por un array de \mbox{\hyperlink{class_fecha_historica}{Fecha\+Historica}} que almacenará en cada posicion los acontecimientos de un año especifico.
