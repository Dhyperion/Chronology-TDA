\doxysubsection{Descripción detallada}
T.\+D.\+A. de un conjuto de acontecimientos en una fecha. 

una instancia {\itshape c} del tipo de datos abstracto ©c \mbox{\hyperlink{class_fecha_historica}{Fecha\+Historica}} es un objeto de los acontecimientos realizados en una fecha, compuesto por un array de strings que almacenará los acontecimientos sucedidos y un valor entero que representa la fecha que sucedierón los acontecimientos.
